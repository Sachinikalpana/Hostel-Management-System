# Hostel Management System

A web-based application built for university mid-project evaluation to streamline student room allocations, check room availability, Payments, logins and logouts, warden logs, hosteler data management etc.

## 🛠️ Tech Stack
* **Frontend:** HTML5, CSS, JavaScript
* **Backend:** PHP
* **Database:** MySQL
